ℹ️ О боте

VideoGenie — это Telegram‑бот, создающий AI‑видео через HeyGen (Fal API).

Возможности:

  • 📝 Текст → видео с аватаром
  • 📸 Фото → оживлённое видео
  • 🌐 Перевод видео с сохранением голоса
  • ⚙️ Интеграция с n8n + Fal API
  • 🚀 Автоматическая отправка в Telegram

Бот работает на базе OpenClaw, n8n и HeyGen.
